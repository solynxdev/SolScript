## Variables
```
// Numbers
int integer		= 0;
float float		= 0.0f;
double double	= 0.0;

// Text
char character	= 'A';
str string		= "Hello, World!";

// Logic
bool boolean	= true;

// General
any any_value	= 0 | 'A' | false

// Miscellaneous
int i;
print(typeof(i)); // prints "null"
```
